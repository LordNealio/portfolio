# NIL — Artwork

Drop artwork images here to feature them on their project pages.

- **Blueface × Salmon P. Chase** → save the piece as
  `blueface-salmon-p-chase.jpg` (JPG or PNG). It appears in that project's
  gallery; missing files are hidden automatically until you add them.
- Then `git add public/art && git commit && git push` — it auto-deploys.
